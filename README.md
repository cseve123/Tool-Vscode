## 配置语言
shift + command + p
输入 config

## 安装code
shift + command + p 
输入 path

## File Utils使用(new,delete,rename, move)
创建文件
shift + command + p
file: New File
创建文件夹
shift + command + p
file: New Folder
指定目录下
command + shift + E （聚焦资源管理区域）
在新建
跳到指定文件
command + P 
输入文件名

## 打开第二个编辑区
command + \

## 选择不同的编辑区
command + 1
command + 2

## 关闭编辑区
command + K + W 关闭了所有文件和编辑区

## vim
移动 hjkl
退出 :q! :wq
删除字符 x
插入 i o a I O A
编辑 vim xxx
删单词 dw de
删(从光标位置)到行末 d$
移动 0（行首）2w 3e(2单词3单词末)
删除多个单词 d3w
删除整行 dd 2dd
撤销 u U()
粘贴 p
替换(光标位置) r R
替换到单词（光标）末尾 ce
组件 c2w
搜索 /  上下n
括号匹配 %
匹配替换 :s/oldXXX/xxx/gc (g是全局，c是提示)
删除文件 :!del xxx || :!rm xxx
复制 y yw y$
下一行末 j$
